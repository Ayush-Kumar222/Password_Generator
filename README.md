<h1>🔐 Password Generator Web Application</h1><br>

<h2>📌 Description</h2>
This project is a Password Generator Web Application built using HTML, CSS, and JavaScript. 
It allows users to generate strong and secure passwords based on customizable criteria such as length and character types.<br>

<h2>🚀 Features</h2>
<ul>
<li><b>Generate random passwords</b></li>
<li><b>Select password length</b></li>
<li><b>Include/exclude:</b></li>
<ol>
<li><b>Uppercase letters</b></li>
<li><b>Lowercase letters</b></li>
<li><b>Numbers</b></li>
<li><b>Special characters</b></li>
</ol>
<li><b>Instant password generation using JavaScript</b></li>
<li><b>Input validation for secure output</b></li>
</ul>
<h2>🛠️ Technologies Used</h2>
<ul>
<li><b>HTML</b></li>
<li><b>CSS</b></li>
<li><b>JavaScript</b></li>
</ul>
<h2>📁 File Structure</h2>
<ul>
<li>index.html   → Structure of the application</li>
<li>style.css    → Styling and user interface</li>  
<li>script.js    → Logic for password generation</li> 
</ul>
<h2>⚙️ Functionality</h2>
<ul>
<li>Users can define the desired password length</li>
<li>Users can choose character types using checkboxes</li>
<li>JavaScript generates a random password based on selected options</li>
<li>Validation ensures at least one option is selected</li>
<li>Password is displayed instantly without reloading the page</li>
</ul>
<h2>🎯 Objective</h2>
To demonstrate JavaScript logic building, DOM manipulation, and form validation by creating a practical and interactive web application.
